Explain the concept of Responsive Design. What does the term mean? What did you discover in your research?

The concept of Responsive Design is using HTML and CSS to automatically resize, hide, shrink, or enlarge a website to make it look good on all devices.

How does Responsive Design relate to web technologies? How can Web pages take advantage of it?

Responsive web design (RWD) is a web design technique that lets web pages work effective on a range of devices and window or screen sizes. Content, design and efficiency across all platforms are important to ensure accessibility and satisfaction.

What techniques, technologies, and patterns did you use in your Responsive design?

My web pages layout is three coloums with main content in the middle. I designed in a way that all text and layout will enlarge or shrink according to the screen size of the device.

What did you learn about building web apps for different devices?

When building a web page, we have to think about the layouts for different devices for user-friendliness.
